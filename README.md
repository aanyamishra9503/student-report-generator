# Student Report Generator System (Python CLI Project)

A simple **Python-based Student Management System** that allows users to create, view, search, update, and delete student academic records using CSV file storage.

---

## Features

- Add new student report
- Calculate total percentage
- Assign grades automatically
- View all student records
- Search student by roll number
- Update student details
- Delete student records
- Logs activity using logging module

---

## Tech Used

- Python
- CSV file handling
- Object-Oriented Programming (OOP)
- Logging module
